Installation

HERMES AGENT peut être installé de plusieurs manières selon votre environnement et vos besoins.

Via pip (Python)

Installation la plus simple via le package Python officiel :

pip install hermes-agent

Via GitHub

Cloner le dépôt pour une installation manuelle :

git clone https://github.com/your-org/hermes-agent.git
cd hermes-agent
pip install -e .

Via Docker

Déploiement facile avec Docker :

docker pull hermes/agent:latest
docker run -d --name hermes-agent -p 8000:8000 hermes/agent:latest

Note : Assurez-vous d'avoir Python 3.11+ et pip installés sur votre système.

Configuration

HERMES AGENT est hautement configurable via des fichiers de configuration et des variables d'environnement.

Fichier de configuration

Le fichier de configuration principal se trouve dans ~/.hermes/config.yaml :

provider: anthropic
model: claude-opus-4.6
max_iterations: 90
session_id: default
platform: cli

Variables d'environnement

Configurez vos clés API via les variables d'environnement :

export ANTHROPIC_API_KEY="***"
export OPENAI_API_KEY="***"

Options principales

Option Description Par défaut
provider Fournisseur de modèle IA anthropic
model Modèle à utiliser claude-opus-4.6
max_iterations Max itérations par session 90
quiet_mode Mode silencieux false

Utilisation

Comment utiliser HERMES AGENT pour vos projets de développement.

Mode CLI

Lancez HERMES AGENT en mode ligne de commande :

hermes --model anthropic/claude-opus-4.6
hermes --tui # Interface TUI interactive

Mode conversation

Utilisez HERMES AGENT comme assistant de développement :

# Exemple de conversation
> Crée une API REST avec FastAPI
> Génère un modèle de données utilisateur
> Teste l'application

Trajectoires et sessions

HERMES AGENT sauvegarde automatiquement vos sessions :

  • Historique des conversations dans ~/.hermes/sessions/
  • Sauvegarde des trajectoires de développement
  • Recherche FTS5 intégrée dans l'historique

Astuce : Utilisez /save pour sauvegarder manuellement une session.

Commandes slash

HERMES AGENT dispose de nombreuses commandes slash pour contrôler son comportement.

Commandes principales

Commande Description
/model Changer de modèle IA
/tools Gérer les outils activés
/skills Installer/gérer les compétences
/save Sauvegarder la session
/clear Effacer le contexte
/help Afficher l'aide

Outils disponibles

HERMES AGENT dispose d'une large gamme d'outils pour le développement.

Outils de base

  • Terminal - Exécution de commandes système
  • Fichiers - Lecture, écriture, recherche et modification
  • Web - Recherche et extraction de contenu web
  • Browser - Automatisation navigateur via Browserbase

Outils avancés

  • Code Execution - Exécution sécurisée de code Python
  • MCP - Intégration Model Context Protocol
  • Delegation - Délégation à des sous-agents
  • Scheduling - Planification de tâches récurrentes

Configuration : Activez/désactivez les outils avec /tools ou dans la configuration.
