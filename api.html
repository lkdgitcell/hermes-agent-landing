Authentification

L'API HERMES AGENT utilise une authentification par clé API. Incluez votre clé dans l'en-tête Authorization de chaque requête.

Authorization: Bearer YOUR_API_KEY

Obtenir une clé API

Vous pouvez générer une clé API depuis votre tableau de bord :

  1. Connectez-vous à votre compte HERMES AGENT
  2. Accédez aux paramètres API
  3. Générez une nouvelle clé
  4. Stockez-la de manière sécurisée

Sécurité : Ne jamais exposer votre clé API publiquement. Utilisez les variables d'environnement.

Conversations

Créez et gérez des conversations avec HERMES AGENT.

Créer une conversation

POST /api/v1/conversations

Crée une nouvelle conversation avec HERMES AGENT.

200 Succès
400 Requête invalide
401 Non autorisé
Paramètre Type Description
model string Modèle IA à utiliser (ex: "anthropic/claude-opus-4.6")
instructions string Instructions système pour l'agent (optionnel)

Envoyer un message

POST /api/v1/conversations/{conversation_id}/messages

Envoie un message dans une conversation existante.

Paramètre Type Description
content string Contenu du message
role string "user" ou "assistant" (défaut: "user")

Exemple cURL

curl -X POST https://api.hermes-agent.com/api/v1/conversations \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{"model": "anthropic/claude-opus-4.6"}'

Sessions

Gérez les sessions de conversation pour persister l'historique.

Lister les sessions

GET /api/v1/sessions

Récupère la liste de toutes les sessions actives.

Récupérer une session

GET /api/v1/sessions/{session_id}

Récupère les détails d'une session spécifique.

Supprimer une session

DELETE /api/v1/sessions/{session_id}

Supprime définitivement une session et son historique.

Outils

Gérez et utilisez les outils disponibles dans HERMES AGENT.

Lister les outils

GET /api/v1/tools

Récupère la liste des outils disponibles et leurs statuts.

Activer/Désactiver un outil

PUT /api/v1/tools/{tool_name}

Active ou désactive un outil spécifique.

Paramètre Type Description
enabled boolean État d'activation de l'outil

Webhooks

Configurez des webhooks pour recevoir des notifications en temps réel.

Créer un webhook

POST /api/v1/webhooks

Enregistre un nouveau webhook pour recevoir des notifications.

Paramètre Type Description
url string URL du endpoint webhook
events array Types d'événements à écouter

Événements disponibles

Événement Description
conversation.created Nouvelle conversation créée
message.received Message reçu par l'agent
tool.called Un outil a été invoqué
session.completed Session terminée
